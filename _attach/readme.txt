首页调用说明

　　所谓首页调用，即在首页特定位置显示出最新的n条留言标题。

两种页面输出方式：

iframe方式：

　　留言标题显示在内嵌框架中，字体样式、背景及配色，都与留言本保持一致。

JavaScript方式：

　　留言标题文字直接在当前页面输出，字体样式及背景，取决于当前页面。如果用户浏览器关闭了JavaScript脚本功能，则文字无法输出。

iframe方式使用方法：

　　在网页代码中插入以下代码：

<iframe width="宽度" height="高度" src="http://留言本位置/tlist.asp?n=显示条数&len=每条最大字数&user=用户名" frameborder="0"></iframe>

JavaScript方式使用方法：

　　在网页代码中插入以下代码：

<script type="text/javascript" src="http://留言本位置/tlist.asp?n=显示条数&len=每条最大字数&js=yes&user=用户名"></script>

目标窗口：

　　当用户点击“查看留言”时，iframe方式默认在包含它的页面窗口中打开，而JavaScript方式则在页面自身的窗口打开。如果您希望在新窗口中打开，请在src属性的末尾加入&target=_blank，两种方式都适用。例如：

<script type="text/javascript" src="http://留言本位置/tlist.asp?n=显示条数&js=yes&user=用户名&target=_blank"></script>

可使用相对路径：

　　如果网站首页和留言本在相同的域名目录中，也可使用相对路径引用tlist.asp。

不限制字数：

　　若不限制标题最大字数，省略len参数即可。

不显示的留言标题：

　　有几种情况的留言标题不会出现在首页调用中：待审核的留言和悄悄话。如果留言本状态为关闭，或者访问者当前IP属于屏蔽范围，同样不会显示。
